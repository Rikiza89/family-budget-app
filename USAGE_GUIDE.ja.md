## 初回セットアップ

### 1. アカウント作成
1. アプリを開いて **「無料で始める」** または **「登録」** をタップ
2. ユーザー名（英数字）を入力
3. パスワード（8文字以上）を入力
4. **「アカウント作成」** をタップ

### 2. 家族情報の設定
- **家族名** — 例: 「田中家」「スミスファミリー」
- **ニックネーム** — 例: 「パパ」「ママ」「太郎」

### 3. カテゴリーの設定
**「デフォルトカテゴリーを使用」** をタップすると以下が自動作成されます：
- **支出**: 食費、外食、日用品、交通費、光熱費、通信費、医療費、娯楽、衣服、保険（積立）、その他
- **収入**: 給料、賞与、副収入、その他収入

カテゴリーはいつでも **設定 → カテゴリー** から追加・編集・削除できます。

### 4. 支払方法の設定
**「次へ」** をタップするとデフォルトが作成されます：
- 現金、クレジットカード、交通系ICカード、PayPay、銀行振込

### 5. 完了！
ダッシュボードが表示されます。さっそく記録を始めましょう 🎉

## 取引の記録

### 支出を記録する
1. 右下の **「+」** ボタンまたは **「支出を記録」** をタップ
2. 金額を入力（クイックボタンも使えます）
3. カテゴリーを選択
4. 支払方法を選択
5. 必要に応じてメモや日付を入力
6. **「保存」** をタップ

### 収入を記録する
支出と同じ手順で、フォーム上部の種類を **「収入」** に切り替えます。

### レシート写真を添付する
取引入力画面で **「レシート（任意）」** をタップし、撮影またはギャラリーから選択します。

### 取引を確認・削除する
**メニュー → 取引履歴** で取引をタップし、**「削除」** をタップします。

## 予算管理

### 予算を設定する
1. **メニュー → 予算管理**
2. カテゴリーを選択（例: 食費）
3. 月額予算を入力（例: 50000）
4. **「予算を追加」** をタップ

### 使用状況の確認
予算は色分けされた進捗バーで表示されます：
- 🟢 **緑色** — 80% 未満
- 🟡 **黄色** — 80% 以上
- 🔴 **赤色** — 予算オーバー

ダッシュボードの **「予算状況」** セクションで一目で確認できます。

### 予算の編集・削除
**メニュー → 予算管理** → 対象予算の **「編集」** または **「削除」** をタップ。

## 貯蓄の記録

アプリでは2種類の貯蓄を管理します：

### 現金貯蓄（直接積立）
給料から貯金口座へ移す、ボーナス貯蓄など。

1. **メニュー → 貯金を登録**
2. 金額と任意のメモを入力
3. **「保存」** をタップ

**効果:** 使えるお金が減る。将来予測グラフの「現金貯蓄」として反映されます。

### 保険型積立（生命保険・学資保険など）
毎月の保険料など、支出でもあり長期貯蓄でもあるもの。

1. **「支出を記録」** → カテゴリー: **「保険（積立）」**
2. 保険料金額を入力
3. **「保存」** をタップ

**効果:** 月次支出にも計上され、かつ長期貯蓄の合計にも加算されます。

### 貯蓄を確認する
**メニュー → 貯蓄サマリー** に以下が表示されます：
- 総貯蓄額（現金貯蓄 + 保険積立）
- 現金貯蓄の履歴
- 保険積立の履歴

## 定期取引

家賃・給料・保険料・サブスクリプションなど、固定的な収支の管理に使います。

### テンプレートを作成する
1. **メニュー → 定期取引**
2. **「➕ 追加」** をタップ
3. 種類・カテゴリー・金額・支払方法を入力
4. **頻度** を設定: 毎日 / 毎週 / 毎月 / 毎年
5. **開始日** と任意で **終了日** を設定
6. 毎月の場合は **毎月の日付** を設定（例: 25日）
7. **「追加」** をタップ

### ワンクリック一括記録（⚡）
1. **メニュー → 定期取引**
2. **「⚡ すべての定期取引を一括記録」** をタップ
3. 対象の取引一覧を確認
4. **「記録する」** をタップ

本日すでに記録済みの取引は自動的にスキップされます（重複なし）。

### 有効/無効の切り替え
テンプレートの **「有効」** スイッチで一時停止・再開できます。

## AI 家計分析

Gemini AI がお客様の取引履歴を読み込み、個別化されたアドバイスを提供します。

### 使い方
1. **メニュー → AI 分析**
2. **分析期間** を選択（1・3・6・12ヶ月 — 3ヶ月がおすすめ）
3. **分析タイプ** を選択：
   - **総合支出分析** — 現状分析、改善提案、節約ターゲットを総合評価
   - **貯蓄最適化** — 貯蓄率の評価、改善方法、将来見通しを分析
   - **予算計画アドバイス** — 理想的な予算配分と月次予算の具体的な提案
   - **カテゴリー深掘り** — 特定カテゴリーの詳細分析
   - **カスタム質問** — 家計について自由に質問できます
4. **「AI 分析を開始する」** をタップ

分析結果は総収入・総支出・収支バランス・貯蓄率のサマリーカードと、AI の詳細なアドバイスとして表示されます。

### 個人 API キーの設定
各ユーザーが個人の Gemini API キーを設定できます：

1. **メニュー → 設定 → AI API キー**
2. キーを貼り付け（`AIzaSy…` で始まります）
3. **「保存」** をタップ

個人キーが優先されます。**「削除」** で解除できます。

> 無料の API キーは [Google AI Studio](https://aistudio.google.com/) で取得できます。

## 将来予測

過去12ヶ月の実績データをもとに、最大60年先の貯蓄推移を予測します。

1. **メニュー → 将来予測**
2. スピナーで予測期間を変更（マウスホイールまたはスワイプ、1〜60年）
3. グラフが自動で更新されます

グラフには現金貯蓄・保険積立・総貯蓄の3本線が表示されます。

## 家族メンバーと招待

### メンバーを招待する
1. **メニュー → 設定 → 家族情報 → メンバー管理**
2. **「新規作成」** をタップ
3. 招待リンクをコピー
4. LINE・メール・SNS などで送信

リンクの有効期限は **7日間**、1回のみ使用可能です。

### 招待リンクから参加する
1. 招待リンクをタップ
2. 「〇〇家に招待されています」と表示されます
3. ユーザー名とパスワードを入力して登録
4. ニックネームを入力
5. **「参加する」** をタップ

### メンバーを削除する
**設定 → メンバー管理** → 対象メンバーの **「削除」** をタップ。

## 通貨設定

1. **メニュー → 設定 → 通貨設定**（または設定画面の 💱 をタップ）
2. ドロップダウンから通貨を選択：
   - ¥ 日本円（JPY）
   - $ 米ドル（USD）
   - € ユーロ（EUR）
   - £ 英ポンド（GBP）
   - ¥ 中国人民元（CNY）
   - ₩ 韓国ウォン（KRW）
   - S$ シンガポールドル（SGD）
   - A$ オーストラリアドル（AUD）
3. **「変更」** をタップ

選択した通貨記号がアプリ全体で使用されます。為替レートは目安です。

## メール通知

指定日数以上記録がない場合にリマインダーメールを送信します。

### 設定方法
1. **メニュー → メール通知**
2. 通知を有効化
3. **「何日間記録がない場合に通知するか」** を設定（例: 3日）
4. **通知先メールアドレス** を入力（1行1アドレス）：
   ```
   papa@example.com
   mama@example.com
   ```
5. **「保存」** をタップ

### 仕組み
- `send_log_reminders` コマンドが最終記録日を確認します
- 設定した日数を超えると、登録されたアドレス全員にメールが送信されます
- 1日に1通まで（スパム防止）

### サーバー設定（管理者向け）
Cron ジョブで毎日実行します：

```bash
# /etc/cron.d/budget-reminders — 毎日朝9時に実行
0 9 * * * youruser /path/to/venv/bin/python /path/to/manage.py send_log_reminders
```

## データエクスポート

指定した月の全取引を CSV 形式でダウンロードできます。

1. **メニュー → 設定 → データエクスポート**
2. 年と月を選択
3. **「CSV ダウンロード」** をタップ

CSV には: 日付、種類、カテゴリー、金額、支払方法、メモ、登録者が含まれます。

Excel や Google スプレッドシートで開いてさらに分析したり、バックアップとして保存できます。

## カテゴリーと支払方法

### カテゴリーを追加する
1. **設定 → カテゴリー → 管理**
2. **「➕ 追加」** をタップ
3. 名前・種類（支出/収入）・アイコン（絵文字）・保険積立かどうかを入力
4. **「追加」** をタップ

### カテゴリーを編集・削除する
「編集」または「削除」をタップします。すでに取引で使用されているカテゴリーは削除できません — 先に該当取引を削除するか別カテゴリーに変更してください。

### 支払方法を追加する
1. **設定 → 支払方法 → 管理**
2. **「➕ 追加」** をタップ
3. 名前と種類（現金・クレジット・ICカード・QR・銀行振込・その他）を入力
4. **「追加」** をタップ

## よくある質問

**家族メンバーは何人まで追加できますか？**
制限はありません。必要に応じて招待リンクを作成してください。

**取引を保存した後に編集できますか？**
現在は削除のみ対応しています。誤った取引を削除して、正しい内容で再登録してください。

**今日すでに記録した定期取引を一括記録したらどうなりますか？**
自動的にスキップされます。重複記録にはなりません。

**将来予測の精度はどのくらいですか？**
12ヶ月の平均値から推計します。あくまで目安として参考にしてください。

**メール通知が届かない場合は？**
SMTP 設定を確認してください。Gmail の場合は2段階認証とアプリパスワードが必要です。

**データは安全ですか？**
データはサーバーに保存されます。重要なデータは定期的に CSV エクスポートでバックアップすることをおすすめします。

**オフラインで使えますか？**
現在はインターネット接続が必要です。PWA 対応のため、将来的なオフライン機能を準備中です。

**どこまで過去のデータを見られますか？**
記録したすべての取引が制限なく保存されます。

**複数の家族グループに所属できますか？**
1アカウントにつき1グループです。別グループに参加したい場合は新しいアカウントを作成してください。

**PC でも使えますか？**
はい。モバイル最適化されていますが、PC ブラウザでも問題なく使用できます。

**予算の色の基準は？**
緑 = 80% 未満、黄 = 80% 以上、赤 = 100% 超過。
